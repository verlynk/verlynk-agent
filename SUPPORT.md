## Security issues

Do **not** file public GitHub issues for security vulnerabilities. Email [contact@verlynk.com](mailto:contact@verlynk.com) with details. See [SECURITY.md](./SECURITY.md).

## Documentation issues

For errors or gaps in this agent documentation repo, open an issue on [verlynk/verlynk-agent](https://github.com/verlynk/verlynk-agent) or submit a pull request. See [CONTRIBUTING.md](./CONTRIBUTING.md).

## Before contacting support

Include:

1. MCP client (Claude, Cursor, ChatGPT, other)
2. Approximate timestamp (UTC)
3. HTTP status or `errorCode` (e.g. `MCP_RATE_LIMIT_EXCEEDED`)
4. Whether using API key or OAuth
5. Whether the org has multiple profiles

Do **not** include API keys, JWTs, or post content with secrets in support requests.
